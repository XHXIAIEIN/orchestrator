# Plan: R79 TLOTP P0 patterns — prompt monorepo infra

## Goal

把 `SOUL/public/prompts/` + `.claude/skills/` 从「扁平 markdown 文档集」升级为「带 @import 组合、循环检测、CI lint 的 prompt monorepo」，同时把 354 行的 `steal/SKILL.md` 作为首个拆分样本落地 3+1 分页菜单规范。**Done 的定义**：`bash scripts/prompt-lint-cycles.sh` 返回 OK；`.github/workflows/` 里新增的 `prompt-lint` job 在一次推送里通过；`.claude/skills/steal/SKILL.md` 从 354 行缩到 <180 行并通过 @import 展开等价于原内容；`SOUL/public/prompts/clarification.md` 含 3+1 分页段落。

## File Map

- `scripts/prompt-lint-cycles.sh` — Create（DFS 循环检测，改编自 tlotp/scripts/import-lint.sh）
- `scripts/prompt-compile.sh` — Create（只做 .md 扁平化到 `.compiled/`，供验证用）
- `.github/workflows/prompt-lint.yml` — Create（新工作流，两 job：cycle-lint + compile-check）
- `.claude/skills/steal/SKILL.md` — Modify（拆分入口，引用 sections/）
- `.claude/skills/steal/sections/01-preflight.md` — Create（Pre-flight 段落）
- `.claude/skills/steal/sections/02-deep-dive.md` — Create（Phase 1 六维扫描 + 深度层）
- `.claude/skills/steal/sections/03-extraction.md` — Create（Phase 2 P0/P1/P2 + Triple Validation + Knowledge Irreplaceability）
- `.claude/skills/steal/sections/04-output.md` — Create（Phase 3 报告模板 + Style Guard + 强制 commit）
- `.claude/skills/steal/sections/05-index-update.md` — Create（Phase 4 + 常见合理化表）

| Risk | Prob | Impact | Mitigation |
|------|------|--------|-----------|
| Claude Code `@skills/` 前缀可能不被识别（原生只认 `@prompts/` 和相对路径？） | Medium | High — 拆分失效 | Step 1 前先查 Claude Code memory 文档确认语法，若不支持则统一用 `@.claude/skills/...` 绝对形式 |
| 拆分后 Claude Code 每次加载 steal skill 会因为 @import 解析慢一帧 | Low | Low | 可接受；或改为 compile 产物作为 skill 发布 |
| 旧 `SKILL.md` 内容过渡期可能被其他 hook 依赖（grep 行号） | Low | Medium | 拆分是纯文本重排，语义等价；step 9 diff 验证 |
